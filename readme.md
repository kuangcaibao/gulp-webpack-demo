模板工程，使用 `gulp + webpack`

## 如何使用

1. git clone 工程

        > git clone https://github.com/kuangcaibao/gulp-webpack-demo.git <your-project-name>

2. 修改工程信息

    通过修改 `package.json`, 把 `demo` 工程信息改为你的工程信息

    在这个目录下，可以对工程中的某些功能进行删除。如当我们只需要 `webpack` 的时候，可以删除掉 `gulp` 相关的依赖

3. 安装依赖

        > npm install
